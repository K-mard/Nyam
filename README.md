# Nyambot - AI 식단 추천 앱

## 📌 프로젝트 개요
Naymbot은 사용자의 나이, 성별, 목표(다이어트/유지/증량),
선호 음식 정보를 기반으로 AI가 하루 식단을 추천해주는
안드로이드 애플리케이션입니다.

본 프로젝트는 AI 프롬프트 설계와 모바일 앱 연동 구조를
중점적으로 구현하였습니다.

---

## 🛠 주요 기능
- 사용자 정보 입력 (나이, 성별, 목표)
- 선호/비선호 음식 입력
- AI 식단 추천 결과 제공
- 추천 결과 화면 분리(ResultActivity)

---

## 🤖 AI 설계 방식
- 실제 GPT API 연동 구조를 구현
- 비용 및 키 보안 문제로 Mock AI 응답 사용
- 프롬프트 설계 문서 별도 제출

---

## 📱 실행 환경
- Android Studio Meerkat
- Kotlin / Jetpack Compose
- Minimum SDK: 24
- Target SDK: 36
